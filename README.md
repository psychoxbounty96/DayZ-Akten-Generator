# DayZ RP – Akten Generator

Kleine clientseitige Web-App für GitHub Pages.

## Nutzung

1. Dateien in ein GitHub-Repository hochladen.
2. GitHub Pages aktivieren.
3. `index.html` öffnen.
4. Subjektnummer, Name und exakt zwei Talente eintragen.
5. PNG herunterladen.

## Keine Speicherung

Die App nutzt nur HTML, CSS und Vanilla JavaScript. Es gibt kein Backend, keine Datenbank und keine Serverkommunikation.

## Koordinaten anpassen

Alle Positionen befinden sich in `script.js` im Objekt `CONFIG.positions`.
Die Koordinaten beziehen sich auf das Bild `assets/akte_blank.png` mit 1536 × 1152 px.
